## Описание

Сервис пользователей и отслеживание их баланса

Стек: JS, ExpressJS, Sequelize, Swagger

Свегер работае на маршруте /api

## Установка

```bash
$ npm install
$ npm i sequelize-cli -g    
$ sequelize-cli db:migrate
$ sequelize-cli db:seed:all
```

## Запуск

```bash
# development
$ npm run start
```
